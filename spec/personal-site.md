# 个人网站需求文档

## 需求描述

为韩方圆（方圆）搭建一个个人品牌网站，面向技术同行、潜在合作者、内容读者等访客，展示个人技术能力、项目经历、开源成果与内容创作，并为后续接入博客、AI 功能等扩展预留空间。

---

## 需求目标

1. **品牌建立**：以简洁、现代的黑白风格呈现个人技术标签（全栈开发 / AI 工程 / 开源 / 内容创作）。
2. **信任传递**：通过关键数字（开源 star、论文、粉丝量）与项目列表，快速建立访客信任。
3. **流量汇聚**：整合 GitHub、X、CSDN、公众号、知识星球等外部链接，形成统一入口。
4. **隐私安全**：不暴露手机号、公司名称等个人敏感信息。
5. **可扩展性**：前端暂不调用后端接口，但保留后端路由与服务层结构，便于后续扩展博客管理、AI 功能等。

---

## 技术方案

### 技术栈

- **框架**：Next.js 15（App Router）
- **语言**：TypeScript
- **样式**：Tailwind CSS + shadcn/ui（基础组件）
- **主题**：双主题（深色默认 / 浅色），通过 `next-themes` 实现
- **国际化**：仅中文，`next-intl` 配置仍保留，语言切换按钮隐藏
- **数据库**：Prisma + SQLite（结构保留，暂不使用）
- **部署**：Vercel（预期）

### 主题设计

- **默认主题**：黑色（dark）
- **可切换**：顶栏保留日/夜主题切换图标
- **配色基调**：深色背景 `#0a0a0a`，亮色文字，亮色品牌强调色（如白色/浅灰）
